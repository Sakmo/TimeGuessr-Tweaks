# TimeGuessr Tweaks

Small tweaks and feature ideas for TimeGuessr to make the game more enjoyable

## Current / planned stuff

- **Year labels**  
  Makes years easier to read at a glance.

- **Auto-submit on timed rounds**  
  If the timer runs out, the round submits automatically.

- **Possible future idea: synchronous challenges**  
  Right now, a challenge just gives both players the same 5 images.  
  The idea here is to make it properly synced, so a new round only starts once **both** players are done with the previous one.
